# El método · skills instalables

El taller no se describe: se instala.

```bash
npm i @alephscript/skills-scriptorium \
  --registry https://npm.scriptorium.escrivivir.co
```

El paquete (hoy `0.6.0`) trae los oficios del taller:

- **swarm-orquestacion** — monta el plan de un mundo y su ciclo de
  obra: briefs, workers, revisión, merge por aceptación.
- **vigilancia** — el vigía read-only: pulso de worktrees, locks y CI,
  gates por rondas, informes dos caras.
- **estacion-viva** — la estación de sesión: watcher, bitácora, pulso.
- **site-web** — este mismo portal: método de copy + publicación Pages.
- **holarquia** — las dos leyes del taller destiladas a skill.

Todo mundo nuevo arranca igual: instalar, `montar-plan`, y el enjambre
trabaja por gates. Así se montó este workspace — y quedó
[acta de cada paso](https://github.com/alephscriptorium-eng/scriptorium/tree/main/plan/SPRINTS/INITIAL-BASE).
