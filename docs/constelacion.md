# La constelación · z·g·s·e·o·a

Seis mundos, seis repos, un mapa. El inventario vivo (tips, estado del
plan, versión del método por mundo) se asienta en
[plan/MUNDOS.md](https://github.com/alephscriptorium-eng/scriptorium/blob/main/plan/MUNDOS.md);
aquí, la lectura de pasillo.

| letra | mundo | en una frase |
| ----- | ----- | ------------ |
| **z** | z-sdk | la letra fundadora: motor y embajador del ecosistema |
| **g** | g-sdk | la ciudad-juego: el tablero donde pasan las cosas |
| **s** | s-sdk | la caja de herramientas techy-devops del taller |
| **e** | e-sdk | el mundo de papel: la imprenta editorial |
| **o** | o-sdk | el puerto: puente de las ciudades hacia el pub |
| **a** | a-sdk | el atril: el scriptorium mismo |

Los mundos se montan como submodules bajo `codebase/` — el índice
apunta, no contiene. Cada mundo gobierna su plan; el workspace declara
la partición en
[plan/PARTICION.md](https://github.com/alephscriptorium-eng/scriptorium/blob/main/plan/PARTICION.md).
