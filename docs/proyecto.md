# Proyecto / DevOps

Flujo del portal: codigo en el repo → build VitePress (`docs:build`) →
verificacion de enlaces (`docs:verificar`) → GitHub Pages.

Las URLs de repositorio, registry, Actions y Pages viven en el footer
(fuente unica del tema; B11). No se duplican aqui.

Este portal se construyó como WP IB-13 del sprint INITIAL-BASE y su
contenido como WP propio; cada cambio pasa gate de vigilancia con
informe público. El día que algo falle, habrá un hash que lo cuente.

Este portal es el **número 0** de un zine que se imprime con actas.
Se montó el mismo día que el método que lo gobierna aprendió a
instalarse solo: un agente que no conocía a nadie fundó este workspace
con el paquete publicado, y su primera ronda de vigilancia no necesitó
corrección. ℵ₀: mundos numerables, historias incontables.
