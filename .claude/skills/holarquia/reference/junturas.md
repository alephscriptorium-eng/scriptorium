# Junturas — protocolo

Una **juntura** documenta la grieta entre dos holones: el material
excedente que hace posible al siguiente. Sin juntura verificable no hay
holón nuevo.

## Forma mínima

1. **Une:** holón A → holón B (ids del registro local).
2. **La grieta:** qué no podía verse desde A y ya empuja hacia B.
3. **Material excedente:** citas y rutas (DS-5); lo no visto =
   `⏳ sin verificar`.
4. **Veredicto de madurez:** ¿hay evidencia bastante para destilar B?
   Si no → acta en rojo; no se inventa B.

## Notaría (describir, no prescribir)

- Describir lo que hay = notaría.
- Prescribir lo que será = spec (otro género; no se cuela aquí).
- Equivalencias parciales se registran como evidencia, **nunca** como
  gatillo automático de destilación.
- Regla de oro: no inventes observaciones.

## Dónde vive

El mundo consumidor elige la carpeta (`REGISTRO_HOLONES` / junturas/).
Este skill no trae dossiers de mundos reales.
