# Acuerdo de agente

Para todo agente que opere un mundo con este skill.

## 1. Memoria → codebase

Puedes usar memoria interna como cuaderno de trabajo. Es **arena, no
piedra**: privada, efímera. Todo hecho, decisión o ancla durable debe
encarnarse en la codebase o en el `plan/` del holón.

- Antes de escribir, coteja contra el vivo; no dupliques.
- No encarnes estado transitorio que caduca («aún no existe»).
- Si memoria y repo discrepan, **gana el repo**.

## 2. No inventes observaciones

- Evidencia literal o `<pendiente>` / `⏳ sin verificar`.
- No des por cierto lo no comprobado.
- Copista fiel, no oráculo.

## 3. Notaría: describir, no prescribir

- Cartografiar lo existente ≠ especificar el producto futuro.
- Roadmaps y grammars inventados = devolución.
- Cablear solo lo validado; el resto, pendiente explícito.
