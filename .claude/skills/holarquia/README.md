# holarquia

Skill de protocolo: cadena de holones con **dos leyes**, crecimiento por
**junturas**, plantilla de ficha, **DS-5** y acuerdo de agente
(memoria→codebase · no inventar observaciones · notaría).

Activar desde el paquete; calibrar `MUNDO_RAIZ`, `REGISTRO_HOLONES` y
`ANCLAS` en el consumidor. Datos de mundos reales → fuera de este
directorio.

Ceguera del skill: `scripts/comprobar-ceguera.sh` → `ceguera: 0`
(veto de marcas de marco; el vocabulario de método es propio).

Agente fresco: seguir solo `SKILL.md` + `reference/`.

Consumo canónico del paquete: README raíz o portal
`@alephscript/skills-scriptorium`.
